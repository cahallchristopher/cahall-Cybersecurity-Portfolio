# PhishGuard Training Scripts

Scripts to train the Q-learning weights. Paste into the PhishGuard popup console to run.

## Recommended Order
1. train-5-reset.js — reset weights to defaults
2. train-1-url-patterns.js — general URL heuristics
3. train-2-homoglyphs.js — brand spoofing detection
4. train-3-redirects.js — redirect chain attacks
5. train-4-tld-risk.js — high risk TLD scoring
6. train-6-bulk-professional.js — professional bulk training with live feeds
